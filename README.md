# 🖼 Image Processing Collage App

A simple Image Processing application built using **Python and OpenCV**.

This app takes a single image, applies multiple image processing techniques, and combines them into a creative collage view.

---

## 🚀 Features

- Grayscale Image
- Binary Image (Thresholding)
- Red Channel Extraction
- Green Channel Extraction
- Blue Channel Extraction
- Mirror Effect
- Automatic Collage Generation

---

## 🛠 Technologies Used

- Python
- OpenCV
- NumPy
